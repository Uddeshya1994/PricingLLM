# LLM-Based Pricing Change Analyzer (Local LLM)

## Overview
This project is an AI-powered pricing change analysis system designed to
detect, analyze, and explain pricing movements in a financial pricing master.
It uses a **local Large Language Model (LLM)** to generate business-friendly
risk assessments and recommendations, ensuring **privacy, zero API cost,
and auditability**.

---

## Key Features
- Pricing delta calculation (absolute & percentage)
- AI-style risk classification (Low / Medium / High)
- Local LLM-based analysis using Ollama (no cloud dependency)
- Structured JSON outputs for system integration
- Automatic Markdown report generation
- Finance & enterprise-ready design

---

## Tech Stack
- Python 3.x
- Ollama (Local LLM)
- Mistral model
- CSV-based pricing master
